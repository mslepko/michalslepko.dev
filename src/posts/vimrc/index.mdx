### Basics

```
<code class="bash">set nocompatible " just in case
```

### Indentation

```
<code class="bash">set expandtab
set shiftwidth=4 " The # of spaces for indenting
set softtabstop=4 " Tab key results in 4 spaces
set tabstop=4

set cindent
set smartindent
set autoindent


set showtabline=2 " Always show tab bar
set foldenable
set foldcolumn=0
set foldmethod=syntax
set foldminlines=0
set foldnestmax=5

set smarttab " At start of line, <Tab> inserts shiftwidth spaces, <Bs> deletes shiftwidth spaces
```

### Misc

```
<code class="bash">syntax on
set background=dark

filetype on
filetype plugin on

set nobackup
set noswapfile
set cursorline
set title " Show the filename in the window titlebar
set backspace=indent,eol,start
set splitbelow " New window goes below
set splitright " New windows goes right

set showcmd
set showmode
set wildmenu

set diffopt=filler
set diffopt+=iwhite
set encoding=utf-8 nobomb
set esckeys

set hidden " When a buffer is brought to foreground, remember undo history and marks
set history=1000 " Increase history from 20 default to 1000

set ttyfast " Send more characters at a given time
set ttymouse=xterm " Set mouse type to xterm

set laststatus=2 " Always show status line
set lazyredraw " Don't redraw when we don't have to

set gdefault " By default add g flag to search/replace. Add g to toggle

set mouse=a " Enable mouse in all in all modes
set noerrorbells " Disable error bells
set nojoinspaces " Only insert single space after a '.', '?' and '!' with a join command
set noshowmode " Don't show the current mode (airline.vim takes care of us)
set nostartofline " Don't reset cursor to start of line when moving around
set nowrap " Do not wrap lines
set nu " Enable line numbers

set shortmess=atI " Don't show the intro message when starting vim
set shell=/bin/sh " Use /bin/sh for executing shell commands
set scrolloff=3 " Start scrolling three lines before horizontal border of window

set report=0 " Show all changes
set ruler " Show the cursor position


au BufNewFile,BufRead *.logic set filetype=perl " .logic files as perl
au BufNewFile,BufRead *.tmpl set filetype=perl " .logic files as perl
```

### Search

```
<code class="bash">set ic
set incsearch
set smartcase " Ignore 'ignorecase' if search patter contains uppercase characters
set hls is
set wrapscan
set hlsearch " Highlight searches
set ignorecase " Ignore case of searches
set incsearch " Highlight dynamically as pattern is typed
```
