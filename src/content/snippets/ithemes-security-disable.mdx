How to disable iThemes security using wp-config.php?


1. Connect to your server using SSH/SFTP or FTP
2. Locate wp-config.php file
3. Add the code below and save the file
4. You should be able to login using URL/wp-login.php

```php
define('ITSEC_DISABLE_MODULES', true);
```
